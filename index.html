Open Big Data Group

This website contains a collection of libraries to be used in processing massive data size in highly distributed and paralleled environment. They are produced by teams at Google and HTC Research Lab , headed by Prof. Edward Chang.

PLDA Parallel Latent Dirichlet Allocation
PSVM Parallelizing Support Vector Machines on Distributed Computers
PFP Parallel FP-Growth for Query Recommendation
PSpectralCluster Parallel implementation of Spectral Clustering
SpeeDO Parallelizing Stochastic Gradient Descent for Deep Convolutional Neural Network
